# react-hls

React компонент с использованием Redux для состояний плеера (воспроизведение, пауза, время трансляции, уровень громкости)

### Установка

```sh
$ npm install
$ node start
```

После запуска сервера проект будет доступен по адресу http://localhost:8080/

### Тесты
```sh
$ npm run test
```
